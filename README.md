# C3 Sound Sketches

Browser-friendly **p5.js** demos that walk through fundamentals of digital sound design—oscillators, modulation, synthesis, sequencing, and sampling—implemented so you can run them locally or paste copies into the [p5.js Web Editor](https://editor.p5js.org/).

## Where to start

- **[`p5-editor/`](./p5-editor/)** — Learning copies with `index.html`, `sketch.js`, and per-sketch READMEs. Open any sketch’s `index.html` in a browser (or use the p5 Web Editor workflow described there).
- **[`script.md`](./script.md)** — Talk / walkthrough notes that match the sketch order and teaching prompts.

For a full list of sketches and short descriptions, see [`p5-editor/README.md`](./p5-editor/README.md).

## Note

These sketches are demonstrations and teaching aids; audio is not heavily mixed—use sensible system volume and headphones if you prefer.
