# Sample and Hold Frequency Steps

This is a p5.js Web Editor learning copy of `sample-hold.html`.

## Concept

Sample a random signal at a fixed rate and hold each value to create stepped pitch movement.

## Use In The p5.js Web Editor

1. Create a new sketch at https://editor.p5js.org/.
2. Replace the default `sketch.js` with this folder's `sketch.js`.
3. Replace the default `index.html` with this folder's `index.html` so p5.sound is loaded.
4. Press the sketch's Start/Play/Trigger control before expecting sound.

## Code Reading Prompt

Try changing MAJOR_SCALE, SH_BASE_AMP, or the default base/range/rate slider values.
