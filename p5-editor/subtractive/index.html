Subtractive synthesis: oscillator → lowpass filter → amp

A classic mono subtractive patch: a harmonically rich oscillator is shaped by an amplitude ADSR and a lowpass filter whose cutoff is also driven by its own ADSR. Try a saw + high resonance for “acid-ish” timbres. Enable the loop to hear it in a simple 16‑step pattern.

Controls guide (what each knob does)

The controls appear in rows above the scope display. Labels are kept short so you can fit a lot on screen; this guide explains what they mean.

Tip: for classic squelch, use saw, raise Res, keep Cut fairly low, and push Env up.

Start audio / Stop audio
Starts/stops the audio graph. Browsers require a user gesture before audio will play.
Trigger
Plays one short note using the current settings (amp + filter envelopes).
Loop + Pattern + BPM
Enables a simple 16‑step sequencer. Pattern selects a looping phrase; BPM sets tempo.
Waveform
Chooses the oscillator waveform: sawtooth (brighter) or square (hollower).
Root note (MIDI)
Root note (MIDI note number). The looper’s patterns are offsets from this root.
Glide
Portamento time (how quickly pitch slides to the next note), useful for basslines.
Amplitude envelope (Attack / Decay / Sustain / Release)
Amplitude envelope: Attack, Decay, Sustain level, Release. Shapes the loudness over time.
Cutoff, Resonance (Q), Envelope amount
Filter cutoff, resonance, and how strongly the filter envelope moves the cutoff (envelope amount).
Filter envelope (Attack / Decay / Sustain / Release)
Filter envelope: Attack/Decay/Sustain/Release for the cutoff motion.
