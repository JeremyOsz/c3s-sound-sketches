Breakbeat slicer: sample & hold on drum breaks

A sample-and-hold idea applied to breakbeats: on each clock step, we sample a random value, hold it, and map that value to a slice index in a drum break from github:switchangel/breaks.

Controls guide
Start / Stop
Starts/stops the clock. Browsers require a click before audio plays.
Break
Chooses one of the remote WAV files from the public breaks repo.
BPM and Rate
Set the clock speed. Rate is measured as steps per quarter note.
Slices
Divides the break into equal regions that can be triggered independently.
Hold drift
Controls how often the held random value changes. Low values repeat slices; high values jump around.
Quantise
Snaps the held value to integer slice positions.
