Advanced FM – simple 2‑operator algorithm

A DX‑style two‑operator patch: one carrier and one modulator. Choose a ratio algorithm, modulation index envelope, and add a feedback loop on the modulator to illustrate how small changes produce very different spectra.

Controls guide (what each control does)

Tip: start with a simple ratio (like 1:1), then raise Modulation index and Feedback slowly — a little goes a long way.

Trigger note
Plays one short note using the current carrier/modulator settings and the envelope controls below.
Algorithm (Carrier:Modulator ratio)
Sets the frequency relationship between the carrier and modulator. Simple ratios give harmonic spectra; odd ratios and feedback can get more metallic.
Modulation index
How strongly the modulator bends the carrier frequency. Higher values make brighter, more complex timbres.
Envelope: Attack / Decay / Sustain
Shapes the modulation index over time (not loudness): attack rises to the peak, decay falls, sustain holds a level while the note plays.
Feedback
Feeds the modulator back into itself. Increases brightness and can create noisy/chaotic spectra at higher settings.
