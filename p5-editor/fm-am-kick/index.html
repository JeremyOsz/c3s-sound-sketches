Pitched percussion: FM + AM (808-style kick)

A single sine carrier is shaped by amplitude modulation (AM envelope) and frequency modulation (FM punch), with a fast pitch-drop envelope. Tune the controls to get classic 808-style kicks.

Controls guide (what each knob does)

Tip: for “more 808”, keep FM decay short (snappy click) and Amplitude decay longer (big tail). For “more tom”, raise Pitch decay and reduce Drop.

Carrier — sound source / waveform
Chooses what generates the raw sound before envelopes/modulation: sine, square, triangle, saw, noise (bandpass-filtered so it still follows pitch), or perlin sample (a looped single-cycle Perlin-noise waveform).
Start audio / Stop audio
Starts (or stops) the WebAudio engine. If audio hasn’t started yet, the synth won’t produce sound.
Trigger kick
Plays one hit: it resets the envelopes, creating a pitch drop + FM “punch” at the start and an amplitude decay for the tail.
auto trigger + Auto (Hz)
When enabled, retriggers the kick at a steady rate. Auto is the rate in hits per second (\(\text{Hz}\)).
pattern + BPM + pattern dropdown
Enables a simple step sequencer. Each pattern is a 16-step grid (16th-notes over one bar). BPM sets the tempo; the dropdown selects a common kick pattern (e.g. 4-on-the-floor, half-time, 2-step).
Base (Hz) — base pitch of the kick
The steady frequency the kick falls toward. This is the “tuning” of the kick body (roughly 30–90 Hz here).
Drop (st) — pitch drop depth
How far above the base pitch the hit starts, in semitones (st). Bigger drop = more dramatic “pew / doof” pitch fall at the start.
Pitch decay (ms) — pitch envelope decay time
How quickly the pitch falls back to Base (in milliseconds). Short = sharp “thump”; long = more “boooom / tom”.
Amplitude decay (ms) — amplitude envelope decay time
How long the kick rings out (tail length). Longer values give a sustained 808-style sub tail; shorter values sound tighter.
FM ratio — FM ratio (modulator-to-carrier frequency)
Sets the modulator frequency as Base × FM r. Small ratios tend to sound rounder; higher ratios make brighter, more complex attack spectra.
FM amount (Hz) — FM amount (frequency deviation)
How strongly the modulator bends the carrier frequency, measured as peak deviation in Hz. Higher values increase “click / knock / bite”.
FM decay (ms) — FM envelope decay time
How quickly the FM “punch” fades out. Short values concentrate the brightness at the very start; longer values smear that brightness into the body.
