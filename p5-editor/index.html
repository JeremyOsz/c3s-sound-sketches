p5 Editor Learning Copies

Navigate the p5.js Web Editor-ready examples in teaching order. Each folder contains index.html, sketch.js, and a short README.

Browser demos Export README
1buffers

Function Generator to Looping Oscillator

Sample one cycle of a function, then loop it fast enough to hear pitch.

2envelopes

Noise Percussion

Turn raw noise into rhythmic percussion with triggered envelopes.

3AM / ring

Amplitude / Ring Modulation

Multiply a carrier by a modulator, from tremolo to fast modulation tones.

4filter + ADSR

Subtractive Synthesis

Shape a bright oscillator with amplitude and filter envelopes.

5harmonics

Wavetable / Additive Synthesis

Build a waveform from sine partials and loop it as an oscillator.

6modulation

Frequency Modulation

Use a second signal to bend oscillator frequency from vibrato to timbre.

72-op FM

Advanced Two-Operator FM

Explore ratio, modulation index envelope, and feedback in a small FM patch.

8kick design

FM + AM Kick

Combine pitch drop, FM punch, and amplitude decay for an electronic kick.

9random

Sample and Hold Frequency Steps

Sample random values at a fixed rate to create stepped pitch movement.

10breaks

Breakbeat Slicer

Use sample-and-hold to choose slices from a remote drum break.

1116-step

Mini Groovebox
