# Mini Groovebox

This is a p5.js Web Editor learning copy of `groovebox.html`.

## Concept

Combine several small synth engines inside a 16-step sequencer, then use the
lead controls to recap AM and sample-and-hold ideas.

## Use In The p5.js Web Editor

1. Create a new sketch at https://editor.p5js.org/.
2. Replace the default `sketch.js` with this folder's `sketch.js`.
3. Replace the default `index.html` with this folder's `index.html` so p5.sound is loaded.
4. Press the sketch's Start/Play/Trigger control before expecting sound.

## Code Reading Prompt

Try changing `PARAM_DEFAULTS.tracks`, `PARAM_DEFAULTS.performance`,
`PARAM_DEFAULTS.leadMod`, or the lead voice defaults.
