# Wavetable / Additive Synthesis

This is a p5.js Web Editor learning copy of `wavetable.html`.

## Concept

Build a waveform from sine partials, write it into a wavetable, and loop it as an oscillator.

## Use In The p5.js Web Editor

1. Create a new sketch at https://editor.p5js.org/.
2. Replace the default `sketch.js` with this folder's `sketch.js`.
3. Replace the default `index.html` with this folder's `index.html` so p5.sound is loaded.
4. Press the sketch's Start/Play/Trigger control before expecting sound.

## Code Reading Prompt

Try changing WT_SIZE or the default partial slider values in createWTUI().
