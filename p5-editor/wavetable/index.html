Wavetable / additive synthesis

Stack a few simple sine partials to draw a custom waveform. We bake one cycle into a wavetable and then loop it as an oscillator. Vary the partial levels to move from sine to richer harmonic or more inharmonic‑ish shapes.

Controls guide (what each control does)
Start / Stop
Starts/stops the wavetable oscillator. Browsers require a user gesture before audio will play.
Base pitch
Sets the pitch of the looping wavetable oscillator.
Partial sliders (1x, 2x, 3x…)
Sets the amplitude of each sine partial (harmonic). Changing these redraws the waveform and rebuilds the wavetable.
